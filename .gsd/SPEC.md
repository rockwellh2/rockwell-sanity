# SPEC.md

> **Status**: FINALIZED (Design Overhaul v2)
> **Goal**: Create a premium, conversion-optimized industrial hydrogen website inspired by the "Glade" Framer template.

## Design Philosophy "The Glade of Hydrogen"
The user finds the current design "shitty". We will pivot to a "Premium, Approachable, Nature-Tech" aesthetic.
- **Inspiration**: Glade (Framer Template). Clean, spacious, high-end.
- **Visuals**:
    - **Typography**:  Sophisticated pairing. (e.g., 'Syne' or 'Outfit' for headers, 'Inter' for body).
    - **Palette**: Deep Forest Green, Hydrogen Blue, crisp White, Soft Greys.
    - **Shapes**:  Move away from "harsh industrial" (sharp corners) to "approachable industrial" (slight rounding, 4px-8px).
    - **Texture**:  Glassmorphism, subtle noise, large hero imagery.
- **Conversion**:  Clear CTAs, sticky booking/contact bars, trust signals prominent.

## Core Features
1.  **Premium Hero**:  Immersive, possibly video or high-quality render.
2.  **Service Cards**:  Beautiful, interactive cards (hover effects, scale).
3.  **System Diagram**:  Interactive or highly polished visual.
4.  **Testimonials/Trust**:  Carousel or grid of partners.
5.  **Clean Navigation**:  Blur-backdrop navbar.

## Tech Stack
-   **Framework**: Next.js 14
-   **Styling**: Tailwind CSS
-   **Animation**: `framer-motion` (for that "Framer" feel).
-   **Background Effect**: Three.js (optimized hydrogen orbital shader - exception to WebGL ban)
-   **Icons**: Lucide React (refined stroke weights).
