# UI Design System

## Goal
Create a consistent visual foundation.

## Instructions
1. Define brand tone.
2. Choose primary, secondary, and neutral colors.
3. Define typography scale.
4. Define spacing and border radius rules.

## Output Format
- Colors
- Fonts
- Spacing scale
- Component rules

## Constraints
- Avoid trendy defaults.
- All tokens must have names.
