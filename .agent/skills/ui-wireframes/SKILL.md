# Wireframes

## Goal
Define layout and hierarchy before visual design.

## Instructions
1. Identify page purpose.
2. Define major sections.
3. Order content by importance.
4. Output a section based layout.

## Output Format
Header  
Hero  
Content blocks  
CTA  
Footer

## Constraints
- No colors.
- No branding.
