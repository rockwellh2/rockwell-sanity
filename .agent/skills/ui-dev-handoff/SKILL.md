# Developer Handoff

## Goal
Ensure engineers can build the design accurately.

## Instructions
1. List pages and components.
2. Provide spacing and typography rules.
3. Describe interactions.
4. Provide asset list.

## Output Format
- Page specs
- Component specs
- Assets
- Interaction rules

## Constraints
- No design language.
- Only implementation details.
