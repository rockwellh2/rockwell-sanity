# UX Copy

## Goal
Create clear and human friendly interface language.

## Instructions
1. Identify user action.
2. Write short direct copy.
3. Avoid jargon.
4. Optimize for clarity.

## Output Format
Element  
Copy  
Purpose

## Constraints
- No marketing tone.
- Must be actionable.
