# Information Architecture

## Goal
Organize content into a logical and usable structure.

## Instructions
1. List all content types and pages.
2. Group them into logical categories.
3. Define primary and secondary navigation.
4. Output a clear site map.

## Output Format
- Top level navigation
- Subpages
- Utility pages
- Page purpose notes

## Constraints
- Navigation must be shallow when possible.
- No duplicate page purposes.
