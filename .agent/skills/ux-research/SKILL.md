# UX Research

## Goal
Extract user needs, business goals, and market context to inform design decisions.

## Instructions
1. Identify the target audience and primary user types.
2. Extract user goals, frustrations, and success criteria.
3. Identify competitors and patterns in the space.
4. Summarize findings as actionable design insights.

## Output Format
- Target Users
- User Goals
- Pain Points
- Competitive Insights
- Design Implications

## Constraints
- Do not invent market data.
- Base insights on provided inputs or clearly stated assumptions.
