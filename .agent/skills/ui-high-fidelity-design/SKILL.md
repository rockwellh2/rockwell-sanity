# High Fidelity Design

## Goal
Create polished visual layouts.

## Instructions
1. Apply design system.
2. Place components.
3. Define spacing and hierarchy.
4. Describe visual intent.

## Output Format

Visual hierarchy  
Key components  
Interaction notes

## Constraints
- Must follow design system.
