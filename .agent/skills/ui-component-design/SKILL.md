# UI Components

## Goal
Design reusable interface components.

## Instructions
1. Identify required components.
2. Define their purpose.
3. Define states like hover, active, disabled.
4. Describe layout and behavior.

## Output Format
Component  
Purpose  
States  
Layout rules

## Constraints
- Components must be reusable.
- No page specific logic.
