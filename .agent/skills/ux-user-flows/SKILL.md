# User Flows

## Goal
Map how users move through the site to accomplish goals.

## Instructions
1. Identify the main user goals.
2. Define the starting point and end state.
3. List each step and decision point.
4. Output a flow in text format.

## Output Format
User Goal  
Step 1  
Step 2  
Decision  
Outcome

## Constraints
- No UI details.
- Only behavioral steps.
